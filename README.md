### 目录结构

> 这里主要介绍下`src`目录，因为开发主要是在这个目录下进行的

src

- Api 定义 api发起请求

- components 项目中通用的组件

- containers 定义页面基本结构，以及注册、登录等弹窗的组件

- lib 各种可能会用到的方法

- model变量类型定义

- pages 页面业务逻辑

- redux state 定义与管理

- routes 定义页面跳转

- statics 静态文件（图片、iconfont 等）

- styles 通用的样式
